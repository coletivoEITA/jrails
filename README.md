= jRails

jRails is a drop-in jQuery replacement for the prototope-rails gem, so that you can still use RJS and Jquery/JqueryUi helpers with the same API as Prototype/Scryptaculous helpers

Support Rails 3.1+

== Resources

Install

Add to your Gemfile

   gem 'eita-jrails'
